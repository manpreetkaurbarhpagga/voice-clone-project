# 🎤 AI Voice Cloning Project

This project is an AI-based Voice Processing System built using FastAPI and Streamlit.
It provides features like Text-to-Speech, Voice Cloning, Speech-to-Text, and PDF-to-Voice conversion.

## 🚀 Features

* Text to Speech (TTS)
* Voice Cloning
* Speech to Text (STT)
* PDF to Voice Conversion

## 🛠️ Tech Stack

* Python
* FastAPI (Backend API)
* Streamlit (Frontend UI)


## ▶️ How to Run the Project

### 1. Install dependencies

pip install -r requirements.txt

### 2. Run FastAPI server

uvicorn main:app --reload

### 3. Run Streamlit UI

streamlit run app.py


## 📁 Project Structure

* main.py → FastAPI backend
* app.py → Streamlit frontend
* voice_engine.py → Voice processing logic
* voice_clone.py → Cloning module
* database.py → Database handling
